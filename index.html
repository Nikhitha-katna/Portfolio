Projects

QuoteBot is an AI-powered web application that helps users generate creative and engaging captions for social media posts. Built using React and Tailwind CSS, the app allows users to input a custom description and select a tone (default, funny, emotional, inspirational, or savage). With the help of OpenAI’s GPT-3.5 model (via OpenRouter), it instantly provides short, catchy captions tailored to the selected tone. The interface is fully responsive and enhanced with smooth animations using Framer Motion for a modern user experience. QuoteBot also includes features like real-time loading feedback and a copy-to-clipboard button with visual confirmation, making it simple and enjoyable to find the perfect caption for any moment.

Redux is used for efficient state management, ensuring smooth data flow and enhancing performance. The application is fully responsive, offering an optimal viewing experience across various devices. It showcases my ability to develop interactive and dynamic web applications while utilizing modern tools and technologies for both front-end and state management, highlighting my proficiency in React and Redux.

The app is designed with a focus on simplicity and usability, offering a seamless experience across devices. This project highlights my skills in React development, local storage management, and creating efficient, user-friendly applications with a clean and modern design.
